# Bundled skills 激活命令一览

以下为仓库内置 skills 的 **`/skill <name> [输入]`** 激活名（`name` 取自各 SKILL.md 的 frontmatter）。  
在聊天里发 **`/skill <激活名>`** 或 **`/skill <激活名> 你的输入内容`** 即可调用对应技能。  
部分技能在支持原生命令的频道会注册为独立斜杠命令（如 `/prose`），名称会规范为 `a-z0-9_`，最多 32 字符。
